# プロジェクト概要 (Project_Overview.md)

## 1. プロジェクト名

ネットワーク異常検知におけるハイブリッドモデル（AutoEncoder + SVM）による検知精度向上に関する研究

## 2. 背景と目的 (なぜやるか)

### 背景
近年、サイバー攻撃の脅威は増大し続けており、ネットワークトラフィックから異常を早期に検知する技術の重要性が高まっています。機械学習を用いた異常検知が主流ですが、多くの手法では「正常」「異常」と正確にラベル付けされた大量の学習データが必要です。しかし、現実にはラベル付けのコストが非常に高く、十分なデータセットを準備することが困難です。

### 先行研究と課題
* [cite_start]**オートエンコーダ (AE) によるアプローチ**: 先行研究（令和4年度 C-2-10）では、AEが「正常な通信」のパターンを学習し、うまく再構成できないデータ（再構成誤差が大きいデータ）を異常として検知する手法が研究されました [cite: 4779, 4782][cite_start]。特に「学習識別再構成」アルゴリズムを用いることで、正常と異常が混在したラベルなしデータからの学習が試みられています [cite: 4779, 4804]。
* **課題**: AEの「再構成誤差」に基づく判断だけでは、巧妙に正常を装った異常（例：再構成は容易だが、パターンの本質が異常）を見逃す可能性があります。

### 本研究の目的
本研究は、オートエンコーダ（AE）の「再構成能力」による視点と、サポートベクターマシン（SVM）による「データパターンの分類能力」の視点を組み合わせた**ハイブリッドモデル**を構築することを目的とします。

これにより、AE単体では見逃してしまう可能性のある異常をSVMが補完的に捉え、ラベルなしデータからでもより高精度かつ堅牢な異常検知システムの実現を目指します。

## 3. 研究アプローチ (何をやるか)

本研究は、先行研究（C-2-10, C-2-22）を基盤とし、以下のステップでハイブリッドモデルを構築・評価します。

1.  **AEモデルの構築**:
    * [cite_start]先行研究（C-2-10）に基づき、「学習識別再構成」アルゴリズムを用いたオートエンコーダを実装します [cite: 4779, 4783]。
    * [cite_start]ラベルなしのNSL-KDDデータセット（正常・異常混合）を用いてAEを学習させます [cite: 4804, 4811, 4812]。

2.  **2種類の判断指標の抽出**:
    * [cite_start]**指標A (再構成誤差)**: 学習済みAEにテストデータを入力し、入力と出力の差（再構成誤差）を計算します [cite: 4782]。
    * **指標B (潜在変数)**: AEがデータを最も圧縮した中間層から、データの特徴を凝縮したベクトル（潜在変数）を抽出します。

3.  **第二の分類器 (SVM) の学習**:
    * [cite_start]先行研究（C-2-22）で用いられたSVMを実装します [cite: 6938, 6939]。
    * 学習データの潜在変数（指標B）をSVMに学習させ、潜在変数の「パターン」が正常か異常かを分類するモデルを構築します。

4.  **ハイブリッド判定と性能評価**:
    * 未知のテストデータに対し、指標A（再構成誤差スコア）と、指標B+SVM（パターン異常スコア）を算出します。
    * これら2つのスコアを統合し、最終的な異常判定を下します。
    * [cite_start]「AE単体」「SVM単体」「ハイブリッドモデル」の3つの性能を、正答率や混同行列を用いて定量的に比較・評価します [cite: 4815, 4824]。
